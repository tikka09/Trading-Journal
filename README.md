# 📈 TradeLog PWA — Setup Guide

Your personal intraday trading journal. Installable on any phone. Works offline. Back up to Google Drive.

---

## 🚀 How to Host (Free) on GitHub Pages — 5 Minutes

### Step 1: Create GitHub Account
Go to **github.com** → Sign Up (free)

### Step 2: Create a New Repository
1. Click the **+** icon → **New repository**
2. Name it: `tradelog`
3. Set to **Public**
4. Click **Create repository**

### Step 3: Upload the 3 Files
Upload these files to your repo:
- `index.html`
- `manifest.json`
- `sw.js`

Click **Add file → Upload files** and drop all 3 files.

### Step 4: Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **main** branch → **/root**
3. Click **Save**
4. Wait 2 minutes — your app is live at:
   **https://YOUR-USERNAME.github.io/tradelog**

---

## 📱 Install on Your Phone (Android)

1. Open **Chrome** on your Android phone
2. Go to: `https://YOUR-USERNAME.github.io/tradelog`
3. Tap the **⋮ menu** → **Add to Home Screen**
4. Tap **Add**
5. TradeLog icon appears on your home screen — it's now an app! 🎉

## 📱 Install on iPhone

1. Open **Safari** on your iPhone
2. Go to: `https://YOUR-USERNAME.github.io/tradelog`
3. Tap the **Share button** (box with arrow)
4. Tap **Add to Home Screen**
5. Tap **Add** — Done! ✅

---

## ☁️ Google Drive Backup (Super Simple)

No complicated setup needed. Here's how:

### To Back Up (after trading each day):
1. Open TradeLog → **Settings** tab
2. Tap **Export to JSON**
3. A file downloads: `tradelog-backup-2026-03-04.json`
4. Move this file to your **Google Drive**

### To Restore (on a new phone):
1. Open Google Drive → download your latest backup JSON
2. Open TradeLog → **Settings** → **Import from JSON**
3. Select the file — all your trades are back! ✅

---

## 📲 Switch to New Phone

1. **Export** your data (Settings → Export to JSON) → save to Google Drive
2. On new phone: open `https://YOUR-USERNAME.github.io/tradelog` in browser
3. Install as app (Add to Home Screen)
4. **Import** your data from Google Drive backup
5. Done — all trades restored! 🎉
